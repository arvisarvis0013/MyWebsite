Project 3: Network Monitoring Tools

Overview

Deployed a network monitoring tool in a Docker container on Debian Linux. Provides real-time visibility into traffic and devices, accessible only on LAN.

Network Setup

Tool runs inside Docker, mapped to LAN IP. Access restricted to trusted LAN devices via firewall rules. VPN with MFA allows secure external connections for approved users only.

Security Setup

Login and MFA required. MAC filtering limits switch connectivity. Wireless networks segregated. VPN with MFA secures remote access.

Notifications

Automated alerts notify administrators of key network events in real time.

Usage

Designed for admins to monitor traffic and receive alerts for issues quickly.
