Project 2: Endpoint Data and Image Backup

1. Infrastructure Setup

Provisioned server hardware with CPU, RAM, and redundant storage (RAID). Installed and configured Type 1 hypervisor (VMware ESXi/Proxmox/Hyper-V). Storage pools and resources optimized for VM performance and reliability.

2. Networking

Created virtual switches and VLANs for management, VM, and storage networks. Applied firewall rules to restrict unauthorized access and control traffic.

3. VM Deployment

Deployed VMs with tailored CPU, memory, and disk allocations. Standardized configurations streamline management and backups.

4. Backup & Restore Strategy

Implemented backup software (Veeam/Altaro) with onsite NAS and offsite/cloud redundancy. Scheduled full and incremental backups with test restores to ensure data reliability.

5. Security Measures

Role-based access controls, network segmentation, encryption of VM disks and backups, and secure protocols in transit. Patch management ensures systems are current.

6. Monitoring & Maintenance

Monitoring tools track host/VM usage. Backup jobs logged with alerts on failures. Routine log reviews and system checks maintain uptime.

7. Disaster Recovery & Documentation

Documented procedures, network design, and security policies. Maintained onsite/offsite copies. Disaster recovery plan includes VM failover to minimize downtime.
